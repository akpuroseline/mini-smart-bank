# 🏦 Python Mini Bank System

## 📌 What It Does

This is a simple command-line banking application built with Python.
It allows users to perform basic banking operations like depositing money, withdrawing funds, and checking their account balance.

The program runs in a loop, so users can continue performing transactions until they choose to exit.

---

## ✨ Features

* 💰 Deposit money into account
* 💸 Withdraw money from account
* 📊 Check current balance
* 🔁 Continuous running system (menu-based)
* ⚠️ Error handling for invalid inputs
* 🚫 Prevents withdrawing more than available balance

### 🔥 Bonus Features (if implemented)

* 📝 Transaction history tracking
* ❌ Prevent negative deposits
* ✅ User-friendly success/error messages

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/python-mini-bank-system.git
cd python-mini-bank-system
```

### 2. Run the program

Make sure you have Python installed, then run:

```bash
python main.py
```
